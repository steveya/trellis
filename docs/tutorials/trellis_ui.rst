trellis-ui
==========

``trellis-ui`` is the experimental companion interface for Trellis.

.. important::

   The UI is documented as experimental because its current shape is more
   task-monitor and knowledge-system oriented than end-user pricing oriented.
   The Python package remains the canonical public workflow today.

Current Focus
-------------

The current UI surfaces emphasize:

- task monitoring
- knowledge-base browsing
- backend pricing endpoints for focused flows such as bond pricing and
  scenario runs

How to Position It
------------------

Document the UI as a companion path, not the canonical path:

- use the homepage to mention it exists
- mark it experimental
- keep the package-level Python tutorials as the source of truth

As the UI becomes pricing-first, the public docs can promote it more
aggressively.
