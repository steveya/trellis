Tutorials
=========

These tutorials are the public, task-oriented path through Trellis.
They focus on package-level APIs, use mock data by default, and call
out live-data, API-key, and experimental paths explicitly.

Tutorial formats
----------------

- text-first walkthroughs for reviewable diffs and versioned docs
- notebook-backed tutorials for interactive exploration

When both exist for the same topic, the text version is the canonical
reference and the notebook is the interactive companion.
